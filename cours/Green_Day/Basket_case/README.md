## Technique
- poignet souple
- main posée sur le chevalet
- refrain :
  - faible amplitude du mouvement du poignet pour jouer les power cords
  - seul le poignet effectue les mouvements
  - Pour s'entrainer :
      - commencer lentement
      - puis accélérer au fur et à mesure
